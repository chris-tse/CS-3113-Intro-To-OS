# diamond.c

A program that accepts a size and an ASCII character as command line arguments then outputs a rhombus of the given size and character.

### Examples

```
$ ./diamond 3 .

  .
 ...
.....
 ...
  .

$ ./diamond 5 '#'
   
    #
   ###
  #####
 #######
#########
 #######
  #####
   ###
    #
```
